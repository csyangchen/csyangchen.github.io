# 马力 VS 扭矩

马力 / power
扭矩 / torque 

一般说的都是发动机输出的功率/马力, 通过传动后到轮上的另外讨论, 会有损耗

- 力 (force): 1牛=1千克物体获得1m/s^2加速度所需的力
- 功 (work): 1焦耳=1牛米, 千瓦时=度
- 功率 (power) / 马力 (horse power): 做功的速度, 千瓦(KW) (瓦=焦耳/秒), 1PS (公制马力单位) = 0.735KW
- 扭矩 (torque): 单位牛米 (不同于功的牛米概念)
- 发动机转速 (rounds per minutes, RPM)

换算
- P = FV
- V = 2*Pi*RPM / 60
- P = T * 2*Pi*RPM / 60

低转速高扭矩
高转速低扭矩

马力机/dynamometer, 马力/扭矩VS转速表, 测量的是轮上马力, 相对发动机马力有损失

# 调教

- 外倾角 / camber: 轮胎和地面法向的夹角
  - 正倾角 / 内八 / positive camber 
  - 负倾角 / 外八 / negative camber 过弯更大的轮胎接触面积提高摩擦力
  - 胎内外温差尽可能小
- 束角 / toe: 轮胎和车侧面的夹角
  - 正束角 / 外八 / toe out / 提高稳定 / 转向不足
  - 负束角 / 内八 / toe in / 转向灵敏 / 转向过度
- 以上两个都会影响轮胎寿命, 优化过弯, 直道负优化
- 后倾角 / 主销倾角 / caster: 避震延长线和地面的夹角 
  - 影响转向力回馈, 0度时转弯不会产生回正分力, 角度越大转向力越大
  - 电子助力消除了直接回馈
